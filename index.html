Welcome to Birdhouse

Birdhouse is a collection of Web Processing Service related components to support data processing in the climate science community. Birdhouse components are written in Python. Many of the OGC/WPS related software comes from the GeoPython project, like PyWPS and OWSLib. Birdhouse is Open Source and released under an Apache License. Read the Birdhouse documentation to get started.
