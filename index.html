Welcome to Birdhouse

Birdhouse is a GitHub organization comprised of Python projects related to Web Processing Services to support data processing in the climate science community.

Birdhouse uses PyWPS and OWSLib from the GeoPython project.

Birdhouse is Open Source and released under an Apache License.
