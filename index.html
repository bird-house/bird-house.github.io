Welcome to Birdhouse

Birdhouse is a collection of OGC Web Processing Service (WPS) related Python components to support data processing in the climate science community. The aim of Birdhouse is to make the usage of WPS easy. Most of the OGC/WPS related software comes from the geopython project. Birdhouse is the home of the following WPS components (birds):

  • Phoenix: a web-based WPS client with ESGF data access
  • Malleefowl: access to climate data (ESGF, ...) as a service and library
  • Flyingpigeon: services for the climate impact community
  • Hummingbird: provides cdo and cfchecker as a service
  • Emu: some example WPS processes
  • Birdy: a WPS command line tool
